# 基于SpringBoot+Vue的宿舍管理系统

#### Description
宿舍管理系统，是前后端分离项目。功能包括：权限管理，菜单管理，项目运行宿舍管理，学生管理，班级管理，宿舍楼管理，各项记录（归寝记录，维修记录，请假记录，晚归记录。
环境配置：Jdk1.8 + IDEA + Mysql 5.7
技术选型： Springboot + Maven + Mybatis + Vue +Node12.16.1
系统角色： 学生、宿管、管理员
